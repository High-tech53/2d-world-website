What is 2D?

2D represents two-dimensional objects that have only height and width. Unlike 3D objects, they lack depth, existing purely on a flat plane. This fundamental concept forms the basis of many art forms, digital designs, and mathematical representations.

Basic 2D shapes form the foundation of geometry

Examples of 2D Shapes

Circle

A perfectly round shape with all points equidistant from the center

Square

Four equal sides and four right angles (90°)

Triangle

Three sides and three angles that sum to 180°

Rectangle

Four sides with opposite sides equal and four right angles

Examples of 2D Art

GameMaker Studio 2

A beginner-friendly yet powerful game development tool for creating cross-platform games from a single codebase. Offers free, Professional ($99.99 one-time), and Enterprise (console support, source access) licenses.

Godot Engine

An open-source game engine supporting both 2D and 3D development, with dedicated tools for 2D design, physics, animation, lighting, and shaders.

Toon Boom Harmony

A professional 2D animation software used in film, TV, games, and advertising, managing the full pipeline from character creation to visual effects.

Blender

Primarily a 3D modeling and animation tool, Blender also supports 2D animation with features like Grease Pencil for drawing and creating complex shapes.

2D Animation

2D animation is used in cartoons, games, and educational videos. It's the traditional form of animation that has evolved from hand-drawn frames to sophisticated digital techniques while maintaining its charm and expressiveness.

Traditional Animation

Hand-drawn frame-by-frame animation that creates fluid movement

Digital Animation

Modern techniques using software to create and manipulate 2D assets
